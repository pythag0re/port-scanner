# Simple Port Scanner – B2 Cybersécurité

## Description
Ce projet est un scanner de ports TCP simple développé en Python.
Il permet de comprendre le fonctionnement des connexions réseau
et l’exposition des services.

(B2 Cybersécurité – Ynov Lyon)

## Fonctionnalités
- Scan TCP basique
- Plage de ports configurable
- Timeout paramétrable
- Sortie lisible en console

## Technologies
- Python 3
- Sockets
- argparse
